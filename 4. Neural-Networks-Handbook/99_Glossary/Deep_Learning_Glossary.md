# Deep Learning Glossary

- neural network: A trainable function made of layers.
- layer: A computation step inside a model, often with parameters.
- neuron/unit: One output value inside a layer.
- parameter: A trainable value updated during learning.
- weight: A parameter that scales an input signal.
- bias: A parameter added to a layer output.
- activation: A nonlinear function applied to layer outputs.
- logit: A raw model score before sigmoid or softmax.
- probability: A value between 0 and 1 used to express confidence.
- loss: A number measuring how wrong a prediction is.
- gradient: The direction and size of change for reducing loss.
- optimizer: The algorithm that updates parameters using gradients.
- epoch: One pass through the training dataset.
- batch: A group of examples processed together.
- overfitting: Fitting training data much better than validation data.
- underfitting: Failing to learn the pattern well enough.
- regularization: A method for reducing overfitting.
- validation: Evaluation on held-out data used for model selection.
- test set: Final held-out data used for the final estimate.
