# PyTorch Glossary

- tensor: A PyTorch array with shape and dtype.
- `Dataset`: An object that returns one example at a time.
- `DataLoader`: An iterable that returns batches from a dataset.
- `nn.Module`: The base class for PyTorch models and layers.
- `forward`: The method defining prediction computation.
- `state_dict`: A dictionary containing model or optimizer state.
- `requires_grad`: A tensor flag telling autograd to track operations.
- `backward`: The call that computes gradients from a loss.
- `torch.no_grad`: A context that disables gradient tracking.
- `model.train`: A mode for training behavior.
- `model.eval`: A mode for evaluation behavior.
- optimizer: A `torch.optim` object that updates parameters.
- scheduler: An object that changes learning rate during training.
